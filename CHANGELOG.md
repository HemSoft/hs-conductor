## [0.1.0] - 2026-01-29

### Added

- Initial release of hs-conductor
- Event-driven multi-agent orchestration with Inngest
- GitHub Copilot SDK integration for AI inference
- Worker types: exec-worker, fetch-worker, file-worker, ai-worker
- Workload definitions: ad-hoc, tasks, workflows
- Admin UI (Electron app) for workload management
- File-based state management in `data/runs/`
